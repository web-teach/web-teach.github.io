#1 Як підключити Tailwind CSS

Tailwind CSS — це utility-first CSS фреймворк, який дозволяє швидко створювати сучасні інтерфейси прямо в HTML за допомогою утилітарних класів.

Спосіб 1: CDN (для навчання та прототипів)

Найпростіший спосіб — підключити через CDN у <head>:

<!DOCTYPE html>
<html lang="uk">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Tailwind CSS</title>
  <script src="https://cdn.tailwindcss.com"></script>
</head>
<body>
  <h1 class="text-3xl font-bold text-blue-600">
    Привіт, Tailwind!
  </h1>
</body>
</html>
Спосіб 2: Встановлення через npm (для проєктів)
# Ініціалізація проєкту
npm init -y

# Встановлення Tailwind CSS
npm install -D tailwindcss

# Створення конфігураційного файлу
npx tailwindcss init

Налаштуйте tailwind.config.js:

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./*.html", "./src/**/*.{html,js}"],
  theme: {
    extend: {},
  },
  plugins: [],
}

Створіть файл src/input.css:

@tailwind base;
@tailwind components;
@tailwind utilities;

Запустіть збірку:

npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch

Підключіть згенерований CSS у HTML:

<link href="./dist/output.css" rel="stylesheet">
Спосіб 3: Tailwind у React (Vite)
npm create vite@latest my-app -- --template react
cd my-app
npm install -D tailwindcss @tailwindcss/vite

Додайте плагін у vite.config.js:

import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})

Додайте у src/index.css:

@import "tailwindcss";

#3 Практичні завдання

Мета: Створити картку профілю користувача з використанням Tailwind CSS.

Вимоги:
  1. Створіть HTML-файл із підключенням Tailwind через CDN
  2. Картка повинна бути по центру екрану
  3. Аватар (зображення) з круглим обрізанням (rounded-full)
  4. Ім'я користувача (text-xl font-bold)
  5. Опис (text-gray-600)
  6. Кнопка "Підписатись" з ефектом hover
  7. Тінь картки (shadow-lg)
Приклад коду:
<div class="min-h-screen flex items-center justify-center bg-gray-100">
  <div class="bg-white rounded-2xl shadow-lg p-8 max-w-sm text-center">
    <img
      src="https://i.pravatar.cc/150"
      alt="Аватар"
      class="w-24 h-24 rounded-full mx-auto mb-4"
    />
    <h2 class="text-xl font-bold text-gray-800">Олександр Коваль</h2>
    <p class="text-gray-500 mt-2">Frontend-розробник | React & Tailwind</p>
    <button class="mt-6 bg-blue-500 hover:bg-blue-700 text-white
      font-bold py-2 px-6 rounded-full transition duration-200">
      Підписатись
    </button>
  </div>
</div>

Мета: Створити адаптивну сітку з картками товарів.

Вимоги:
  1. Мобільний: 1 колонка
  2. Планшет (md): 2 колонки
  3. Десктоп (lg): 3 колонки
  4. Кожна картка містить: зображення, назву, ціну, кнопку "Купити"
  5. Ефект підняття картки при наведенні (hover:-translate-y-2)
  6. Плавні переходи (transition duration-300)
Шаблон:
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6 p-8">

  <!-- Картка товару -->
  <div class="bg-white rounded-xl shadow-md overflow-hidden
    hover:-translate-y-2 hover:shadow-xl transition duration-300">
    <img src="https://picsum.photos/400/250" alt="Товар"
      class="w-full h-48 object-cover">
    <div class="p-6">
      <h3 class="text-lg font-semibold text-gray-800">Назва товару</h3>
      <p class="text-gray-500 mt-2">Короткий опис товару</p>
      <div class="flex items-center justify-between mt-4">
        <span class="text-2xl font-bold text-green-600">₴ 999</span>
        <button class="bg-green-500 hover:bg-green-700 text-white
          font-bold py-2 px-4 rounded-lg transition duration-200">
          Купити
        </button>
      </div>
    </div>
  </div>

  <!-- Повторіть картки для кожного товару -->
</div>

Мета: Створити повноцінну лендінг-сторінку для вигаданого продукту.

Структура сторінки:
  1. Навбар — логотип + навігація (flex justify-between)
  2. Hero-секція — заголовок, підзаголовок, CTA-кнопка, фонове зображення або градієнт (bg-gradient-to-r from-blue-500 to-purple-600)
  3. Секція "Переваги" — 3 картки в ряд із іконками (grid grid-cols-1 md:grid-cols-3)
  4. Футер — копірайт, посилання
Приклад Hero-секції:
<section class="bg-gradient-to-r from-blue-600 to-purple-700
  text-white py-20 px-8">
  <div class="max-w-4xl mx-auto text-center">
    <h1 class="text-4xl md:text-6xl font-bold mb-6">
      Створюй швидше з Tailwind
    </h1>
    <p class="text-xl md:text-2xl text-blue-100 mb-8">
      Utility-first CSS фреймворк для сучасної веб-розробки
    </p>
    <a href="#" class="bg-white text-blue-600 font-bold py-3 px-8
      rounded-full text-lg hover:bg-blue-50
      transition duration-300 inline-block">
      Почати безкоштовно
    </a>
  </div>
</section>
Додаткові вимоги:
  • Сторінка повинна бути повністю адаптивною
  • Використовуйте hover ефекти для всіх інтерактивних елементів
  • Додайте плавні переходи (transition)
  • Створіть мінімум 6 унікальних карток, кнопок або секцій
