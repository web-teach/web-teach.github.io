#2 Створення першого проєкту на React

  • npm create vite@latest my-app --template react - створення нового проєкту з ім'ям my-app за допомогою інструменту збірки Vite.
  • cd my-app - перехід у папку проєкту.
  • code . - відкрити VSCode у поточній папці (за потреби).
  • npm install - встановлення залежностей.
  • npm run dev - запуск локального сервера.

  • src/main.jsx - точка входу у застосунок.
  • src/App.jsx - головний компонент.
  • src/index.css - файл стилів.
