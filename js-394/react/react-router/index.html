#2 Основні концепції React Router

React Router — це стандартна бібліотека для маршрутизації в React-застосунках. Вона дозволяє:

  • Створювати Single Page Applications (SPA) з навігацією між різними сторінками
  • Синхронізувати UI з URL-адресою
  • Використовувати декларативний підхід до маршрутизації
  • Передавати параметри через URL
  • Керувати історією браузера

Основні переваги:

  • Покращене UX (без перезавантаження сторінки)
  • Швидка навігація між сторінками
  • Підтримка закладок та історії браузера
  • SEO-дружній підхід

Для встановлення React Router DOM (версія для веб-застосунків) виконайте команду:

npm install react-router-dom

Примітка: React Router має кілька пакетів:

  • react-router-dom — для веб-застосунків (браузер)
  • react-router-native — для мобільних застосунків (React Native)
  • react-router — базова бібліотека (включена в обидва пакети вище)

1. BrowserRouter (або RouterProvider)

Обгортає весь застосунок і забезпечує контекст маршрутизації:

import { BrowserRouter } from 'react-router-dom';

function App() {
  return (
    <BrowserRouter>
      {/* Ваш застосунок */}
    </BrowserRouter>
  );
}
2. Routes та Route

Routes — контейнер для всіх маршрутів. Route — визначає окремий маршрут:

import { Routes, Route } from 'react-router-dom';

function App() {
  return (
    <Routes>
      <Route path="/" element={<Home />} />
      <Route path="/about" element={<About />} />
      <Route path="/contact" element={<Contact />} />
    </Routes>
  );
}
3. Link

Компонент для навігації без перезавантаження сторінки:

import { Link } from 'react-router-dom';

function Navigation() {
  return (
    <nav>
      <Link to="/">Головна</Link>
      <Link to="/about">Про нас</Link>
      <Link to="/contact">Контакти</Link>
    </nav>
  );
}
4. NavLink

Розширена версія Link з підтримкою активного стану:

import { NavLink } from 'react-router-dom';

function Navigation() {
  return (
    <nav>
      <NavLink 
        to="/" 
        className={({ isActive }) => isActive ? 'active' : ''}
      >
        Головна
      </NavLink>
    </nav>
  );
}

React Router дозволяє передавати динамічні параметри через URL:

import { Routes, Route } from 'react-router-dom';

function App() {
  return (
    <Routes>
      <Route path="/user/:id" element={<UserProfile />} />
      <Route path="/product/:category/:id" element={<Product />} />
    </Routes>
  );
}

Для отримання параметрів використовується хук useParams:

import { useParams } from 'react-router-dom';

function UserProfile() {
  const { id } = useParams();
  
  return <h1>Профіль користувача #{id}</h1>;
}

function Product() {
  const { category, id } = useParams();
  
  return (
    <div>
      <h1>Товар #{id}</h1>
      <p>Категорія: {category}</p>
    </div>
  );
}

Хук useNavigate дозволяє програмно переходити між сторінками:

import { useNavigate } from 'react-router-dom';

function LoginForm() {
  const navigate = useNavigate();
  
  const handleSubmit = (e) => {
    e.preventDefault();
    // Логіка входу...
    
    // Перехід на головну сторінку
    navigate('/');
    
    // Перехід назад
    // navigate(-1);
    
    // Перехід вперед
    // navigate(1);
    
    // Перехід з заміною історії
    // navigate('/dashboard', { replace: true });
  };
  
  return <form onSubmit={handleSubmit}>...</form>;
}

React Router підтримує вкладені маршрути для створення складних макетів:

import { Routes, Route, Outlet } from 'react-router-dom';

function App() {
  return (
    <Routes>
      <Route path="/" element={<Layout />}>
        <Route index element={<Home />} />
        <Route path="about" element={<About />} />
        <Route path="products" element={<Products />} />
      </Route>
    </Routes>
  );
}

function Layout() {
  return (
    <div>
      <header>
        <nav>{/* Навігація */}</nav>
      </header>
      <main>
        <Outlet /> {/* Тут відображаються вкладені маршрути */}
      </main>
      <footer>{/* Футер */}</footer>
    </div>
  );
}

Для обробки неіснуючих маршрутів використовується спеціальний шлях:

import { Routes, Route } from 'react-router-dom';

function App() {
  return (
    <Routes>
      <Route path="/" element={<Home />} />
      <Route path="/about" element={<About />} />
      <Route path="*" element={<NotFound />} />
    </Routes>
  );
}

function NotFound() {
  return (
    <div>
      <h1>404 - Сторінка не знайдена</h1>
      <Link to="/">Повернутися на головну</Link>
    </div>
  );
}

#3 Практичне завдання

Мета: Створити простий багатосторінковий застосунок з навігацією.

Кроки виконання:
  1. Встановіть React Router: npm install react-router-dom
  2. Створіть сторінки: Home, About, Products, Contact, NotFound
  3. Налаштуйте маршрутизацію в App.jsx
  4. Створіть компонент Navigation з посиланнями
  5. Додайте сторінку з динамічним параметром (наприклад, /product/:id)
  6. Реалізуйте обробку 404 помилки
  7. Додайте активний стан для навігації (NavLink)
Приклад структури:
// App.jsx
import { BrowserRouter, Routes, Route } from 'react-router-dom';
import Navigation from './components/Navigation';
import Home from './pages/Home';
import About from './pages/About';
import Products from './pages/Products';
import ProductDetail from './pages/ProductDetail';
import Contact from './pages/Contact';
import NotFound from './pages/NotFound';

function App() {
  return (
    <BrowserRouter>
      <Navigation />
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
        <Route path="/products" element={<Products />} />
        <Route path="/products/:id" element={<ProductDetail />} />
        <Route path="/contact" element={<Contact />} />
        <Route path="*" element={<NotFound />} />
      </Routes>
    </BrowserRouter>
  );
}

Створіть макет з загальною структурою (header, footer) та вкладеними маршрутами:

// App.jsx
function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Layout />}>
          <Route index element={<Home />} />
          <Route path="about" element={<About />} />
          <Route path="blog">
            <Route index element={<BlogList />} />
            <Route path=":postId" element={<BlogPost />} />
          </Route>
        </Route>
      </Routes>
    </BrowserRouter>
  );
}

// Layout.jsx
import { Outlet } from 'react-router-dom';

function Layout() {
  return (
    <>
      <header><Navigation /></header>
      <main><Outlet /></main>
      <footer>© 2025 My App</footer>
    </>
  );
}
