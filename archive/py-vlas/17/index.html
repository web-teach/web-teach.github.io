Команди терміналу для настройки віртуального середовища

  • python -m venv .venv - створити нове віртуальне середовище, якщо не було створено раніше (.venv - назва середовища).
  • .venv\Scripts\activate - активація віртуального середовища, якщо немає (.venv) на початку команди в терміналі.
  • python -m pip install flask - встановлення Flask.
  • python app.py - запуск Flask-додатку.
