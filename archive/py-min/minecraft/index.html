#1 Налаштовуємо Python у Minecraft
Завантажити Java Minecraft Python API & Spigot

Для того щоб запускати програми на Python для Minecraft потрібно:

  1. Мати встановлені наступні програми:
    • Minecraft
    • Python 3.7
    • Java 8
    • Minecraft Python API & Spigot
  2. Дозволити запуск сервера без інтернету. Для цього у папці "C:\Users\User\Documents\Minecraft Python\Minecraft Tools\server" змінюємо файл "server.properties". Потрібно задати значення параметра online-mode=false
  3. Запустити Minecraft-сервер за допомогою файлу Start_Server із папки Minecraft Tools.
  4. Запустіть Minecraft і виберіть пункт меню Мережева гра (Multiplayer). Натисніть на кнопку Додати (Add Server). У рядку введення "Назва сервера" (Server Name) назвіть ваш сервер "Minecraft Python World", а в рядку "Адреса сервера" (Server Address) введіть localhost. Потім клікніть Готово (Done). Зробіть подвійний клік по Minecraft Python World, і відкриється світ, створений сервером Spigot.
  5. Створюєте код і програмуєте свій світ Minecraft!

#2 Основні команди Python у Minecraft

  • Для того щоб підключитися до світу Minecraft у Python потрібно:

    # Підключаємося до Minecraft
from mcpi.minecraft import Minecraft
mc = Minecraft.create()

  • Для надсилання повідомлень у чат гри:

    mc.postToChat("Hello world!")

    Бонус

    Ви можете додати паузу у програму, використовуючи модуль time. 

    import time # Додайте цей рядок на початок програми
time.sleep(3) # 3-х секундна пауза

  • Робота з координатами

    pos = mc.player.getTilePos() - отримати координати гравця

    mc.player.setTilePos(x, y, z) - задати координати гравцю (телепортувати гравця)

    mc.player.setPos(x, y, z) - задати точні координати гравцю

    Бонус

    Ви можете задати випадкові координати використовуючи модуль random. Наприклад, задати випадкову координату x на проміжку від 0 до 100:

    import random # Додайте цей рядок на початок програми
x=random.randint(0,100)

  • Побудова блоків

    mc.setBlock(x, y, z, blockType) - побудувати блок типу blockType (ідектифікатор блоку) в координатах x, y, z.

    mc.setBlocks(x, y, z, x + width, y + height, z + length, blockType) - побудувати кубіоїд із блоків типу blockType (ідектифікатор блоку) в координатах x, y, z розмірами width, height, length.

    mc.setBlocks(x, y, z, x + width-1, y + height-1, z + length-1, blockType) - додатково побудувати порожній кубіоїд трохи меншого розміру всередині звичайного. Для цього змінній blockType задаємо значення 0 - повітря
