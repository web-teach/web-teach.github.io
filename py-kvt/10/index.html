#1 Робота з текстовими файлами у Python

ЗАВДАННЯ 1

Створи текстовий файл data.txt з довільним вмістом (наприклад, 3-5 рядків тексту). Напиши програму, яка зчитує цей файл та виводить його вміст на екран.

Підказка: Використай функцію open з режимом 'r' та метод read() або readlines(). 
# Приклад:
# Вміст файлу data.txt:
Привіт!
Це тестовий файл.
Python — класний!

# Вивід програми:
Привіт!
Це тестовий файл.
Python — класний!
Розширення (за бажанням):
  • Вивести кількість рядків у файлі.
  • Вивести кількість слів у файлі.

ЗАВДАННЯ 2

Напиши програму, яка запитує у користувача декілька рядків тексту (поки не буде введено порожній рядок) і записує їх у файл output.txt.

Підказка: Використай цикл while та функцію open з режимом 'w' або 'a'. 
# Приклад:
Введіть рядок: Мене звати Оля
Введіть рядок: Я вивчаю Python
Введіть рядок: 
# (порожній рядок — завершення)
# Вміст файлу output.txt:
Мене звати Оля
Я вивчаю Python
Розширення (за бажанням):
  • Додати можливість вибору режиму запису: перезаписати файл чи дописати в кінець.
  • Порахувати кількість символів, записаних у файл.

ЗАВДАННЯ 3

Є файл numbers.txt з числами, записаними через пробіл. Напиши програму, яка зчитує цей файл, знаходить суму всіх чисел і виводить результат.

# Приклад:
# Вміст файлу numbers.txt:
5 10 15 20
# Вивід програми:
Сума чисел: 50
Розширення (за бажанням):
  • Знайти середнє арифметичне чисел.
  • Вивести найбільше та найменше число.

ЗАВДАННЯ 4

Створи програму, яка зберігає дані у форматі JSON. Запитай у користувача ім'я, вік та місто, збережи ці дані у словник, а потім запиши у файл user.json у форматі JSON.

Підказка: Використай модуль json та функцію json.dump(). 
# Приклад:
Введіть ім'я: Олег
Введіть вік: 15
Введіть місто: Київ
# Вміст файлу user.json:
{"name": "Олег", "age": 15, "city": "Київ"}
Розширення (за бажанням):
  • Зчитати дані з user.json і вивести їх у зручному вигляді.
  • Додати можливість зберігати декілька користувачів у список у файлі.
