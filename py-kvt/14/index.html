ЗАВДАННЯ ПРОЄКТУ

Нас попросили допомогти з розробкою бази даних для інтернет-магазину. Наше завдання — організувати керування базою даних та взаємодію з нею через консоль.

Виконайте наступні кроки:

Створити базу даних shop.db та таблиці Товари, Категорія, Замовлення, Кошик:

  • Таблиця Товари містить поля: id, назва, опис, ціна, id_категорії
  • Таблиця Категорія містить поля: id, назва
  • Таблиця Замовлення містить поля: id, дата, товари, ціна
  • Таблиця Кошик містить поля: id, товари, загальна ціна

Реалізувати клас DatabaseManager для взаємодії з базою даних, який буде містити методи для маніпуляції базою даних, наприклад:

  • Отримання всіх товарів з бази даних
  • Отримання всіх товарів певної категорії
  • Пошук по базі товарів
  • Створення нових записів та видалення в таблицях Замовлення та Кошик

Реалізувати клас OnlineStore для керування інтернет-магазином через консоль:

  • Перегляд всіх товарів
  • Перегляд детальної інформації про товар
  • Перегляду товарів за категорією
  • Пошук товару за назвою
  • Додавання товарів до кошика
  • Оформлення замовлення

Перевірити роботу всіх методів класів DatabaseManager та OnlineStore шляхом створення та взаємодії з об'єктами через консоль.

Тестувати додавання нових категорій, товарів, оформлення замовлень та додавання товарів до кошика.
